Description of Bored and CFA piles axial capacity calculator

This program uses CPT data to calculate the axial capacity of bored piles using the method described in:

Doan, L.V. and Lehane, B.M. (2021). "CPT-Based Design Method for Axial Capacities of Drilled Shafts and Auger Cast-in-Place Piles." J. Geotech. & Geoenv. Engrg., ASCE, 147(8). DOI: 10.1061/(ASCE)GT.1943-5606.0002542
